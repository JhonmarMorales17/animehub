# AnimeHub 🎌

Explorador de anime construido con HTML, CSS y JavaScript vanilla, utilizando la API de Jikan (MyAnimeList).

## ✨ Características

- 🏠 **Página principal** con los animes más populares
- 🔍 **Búsqueda** de animes por nombre
- 📱 **Diseño responsive** para móviles y desktop
- ⚡ **SPA** (Single Page Application) sin recargas
- 🎨 **Componentes web** personalizados
- ✨ **Estados de carga** y manejo de errores

## 🛠️ Tecnologías

- HTML5
- CSS3 (Flexbox, Grid, Variables)
- JavaScript Vanilla (ES6+)
- [Jikan API](https://jikan.moe/) (API no oficial de MyAnimeList)
